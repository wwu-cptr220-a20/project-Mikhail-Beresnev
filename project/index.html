Issue a Code

Issue a new code for a completely new chat room. After recieving the code, send this code to the people you may wish to talk to. Additionally, enter the expiration date and time for the code in the case when at least two people have not joined the chat room (the minimum of the issuer and a different person). The default time is set to 3 days. Finally, the chat room will be deleted after 1 week of no use.
